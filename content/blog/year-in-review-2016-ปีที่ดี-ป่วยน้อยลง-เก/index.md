<h3>         เป็นปีที่ทำงานหนักแล้วรู้สึกว่าครบปีไวมากๆ</h3>
ประโยคที่พูดไป ปลายปีที่แล้วยังคุ้นหูอยู่เลย และเป็นปีหนึ่งในชีวิตนี้เลยที่รู้สึกว่าเก่งขึ้นมาก และเพราะสนุกกับมันมากขึ้นก็เลยทำให้รู้สึกว่ามันไว และพยายามลุกหนีออกจากวง Coder ใจจริงไม่ได้อยากหนีหายไปไหนหรอก แค่รู้สึกว่าต้องโยกมาทำอย่างอื่นบ้าง ไม่งั้นเดี้ยวอย่างอื่นเสียหายหมด ต้องมาลุยงานด้าน Management จริงๆจังๆ ซักที
<h3>        ไอ้ที่กลัวสุดตอนเริ่มปีนี้ก็คือ</h3>
ผมกำลังเปลี่ยนจาก Developer มือดีคนนึง มาเป็น Manager ห่วยๆ คนหนึ่งหรือเปล่า เพราะไม่เคยโตมาในสาย Management เลย แต่ก็พยายามเรียนรู้เทคนิควิธีคิดจากคนอื่นๆ มา จำจด และลอง แต่มันก็ผ่านมาได้สำหรับปีนี้ โดยเหมือนจะเป็นครั้งแรกที่ลงมาจัดการทีมพัฒนาซอฟท์แวร์ทั้งหมดทุกตัวของบริษัทด้วยตนเอง ถามว่าสนุกไหม ต้องถามทีม แต่ส่วนตัวพอใจระดับนึง แต่ยังมีอะไรต้องทำอะไรอีกเยอะ
<h3>        ปีนี้สิ่งที่ดีสุดคือ</h3>
ผมพยายาม จัดเสาร์และอาทิตย์ให้ครอบครัวมากที่สุดเท่าทีจะทำได้ และรู้สึกดีที่ได้ให้เวลากับลูกและครอบครัวเต็มที่ทั้งสองวันอาจจะมีบ้างที่ติดงานแต่ก็ยังได้อยู่บ้านด้วยกันแบบกิจกรรมครอบครัวตามภาษาคนชอบอยู่บ้าน

<h3>เป็นปีที่ได้เรียนรู้ Agile ได้ใช้มันจริงๆ และรู้ว่ายากกว่าที่คิดเยอะ</h3>
ไปอบรมมา เอากลับมาใช้  10 รอบ Sprint  ได้ปรากฏว่าเละครับ ก็คนไม่เคยนี่หว่าแถมไม่มี Scrum Master ต้องสวมบทครอบ อีกที ลำบากแท้ แต่ก็รู้ว่า จริงๆ แล้วมันก็แค่เรื่องพฤติกรรม การทำงาน หลังๆ ชักแม่น วิธีการ แต่พลังน้อยลงไป ต้องเข้มข้นอีกนิดปีหน้า ทีมจะได้เก่งขึ้นอีก

&nbsp;
<h3>ปีแห่งการขยายสาขา</h3>
อย่างที่รู้กันเราเปิดบริษัทเดิมทีสาขามันไกล้ๆ กันยิง Wifi ก็เชื่อมแล้วแต่รอบนี้ไกลกันไหน่อย จะเรียกว่าสาขาหรือเรียกสำนักงานใหญ่ไม่รู้ ทั้งโทรศัพท์ สายใน สายนอก Server ต่างๆ Application ทั้งหมด ต้องเชื่อมกันได้ทั้งหมด เป็นครั้งแรกที่ผมทำ WAN สาขาให้บริษัทตัวเอง หลังๆ มีระบบ Conference อีก ทั้งหมดใช้ Open Source จึงเป็นอีกปีที่ยอดเยี่ยมมากกับการศึกษา Open Source ชอบๆ

<h3>เป็นการกลับบ้านเกิดที่แท้จริงในรอบสิบปี</h3>

<h3>เป็นปีที่กล้าสร้าง แนวคิดใหม่ๆออกมา และกล้าทดลองบางอย่างบน Production</h3>
Docker นี่ หลงเข้ามาเรียนรู้ไม่นาน ก็ไม่รู้อะไร ดลบัลดาลใจ ให้ใช้เลยทั้นที แต่ส่วนตัวชอบมาก อะไรแบบนี้ เหมือนตอบโจทย์ที่เราหามานาน ได้ดีระดับนึง พอ ขยับมาศึกษา Compose เห้ยมันดีนี่หว่า ไม่ลำบากเวลา Setup กำลังจะเอาขึ้น Production Docker 1.12 ก็ออกมาพอดีได้นี่อะไรว่า Swarm Mode เทห์มาก สุดท้ายมาลงเอยกับ Swarm ชอบใจมาก ส่วนอีกเรื่องเจอปัญหา Angular 1 ช้าบน Mobile เหรอกิน CPU เหรอ เราเลยคิดว่าจะทำยังไงจะไป React ไหม หรือจะยังไงในที่สุดก็ใช้วิธี ไม้ง้อ จับแม่ง Compile รวดเดียว Append เลย Angular รุ่น Zero $$Watch ไวใช่เล่น แต่จริงๆ แล้วคงไม่ถึง 0 หรอกแต่จะพยายามทำให้ได้น้อยที่สุด
<h3>อยากขอบคุณใครบ้างในปีนี้</h3>
แน่นอนครับมีหลายคน กลุ่มแรกเลยก็ครอบครัวแหละ เขาช่วยเราได้มากทีเดียวที่ทำให้ไม่ต้องคิดอะไรมากเรื่องของพวกเขา กลุ่มที่สอง Developer ทั้งหมด ของ iTopPlus ครับ เก่งขึ้นตามที่พี่สัญญาไว้ตอน มค. 2016 ไหม อันนี้ถ้าถามพี่ก็คงบอกว่าดีขึ้น กลุ่มที่สาม พนักงาน iTopPlus ทุกคนที่ช่วยกันแก้ปัญหาต่างๆ ให้ลูกค้า แน่นอนคำชมเชยมันคงยังไม่กลับมาในเร็ววัน เราจะพยายาม ทำให้มันดีขึ้น ให้ได้กว่านี้ในปีหน้าผมสัญญา
